# Hash Tables

## Description
This project implements a basic hash table in C using chaining for collision handling.

## Files
- 0-hash_table_create.c: function to create a hash table
- hash_tables.h: header file with structures and prototypes

## Compilation
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o output

## Usage
Include "hash_tables.h" and use hash_table_create() to initialize a hash table.
